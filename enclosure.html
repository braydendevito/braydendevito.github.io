Electronics Enclosure

3D Model / Exploded View

For this project, I combined knowledge of electronics, and mechanical design in one project. This is a controller box and a waterproof yard node for an automated, smart sprinkler system. This project was especially fun to work on because I got to implement my mechanical designs in a green product that would be responsible for helping people across the world save water, and keep a healthy lawn.
